# React learning [In progress]

### Usage Python

- run each task with Python server:

```
python3 -m http.server
```

- open task in the browser:

```
http://localhost:8000/
```

### Usage React

- create React project:

```
npx create-react-app app-name
cd ./app-name
npm start
```
